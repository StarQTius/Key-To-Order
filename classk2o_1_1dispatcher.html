Detailed Description

template<size_t N>
class k2o::dispatcher< N >

Order containers able to unserialize byte sequence serialized by an 'ikey' object.

A 'dispatcher' object is constructed from a 'keyring11' object and is able to unserialize a payload serialized by an 'ikey' object created from the same 'keyring11' object. When it happens, the 'dispatcher' object calls the function associated with the 'ikey' object, forwarding the arguments from the payload to the function. The functions are internally held as 'order' objects.

Template Parameters
Nthe number of functions held by the 'keyring11' object
